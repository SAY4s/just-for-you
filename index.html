⚙️ تنظیمات ارسال

این بخش فقط برای خودته (فرستنده). فقط در حافظه‌ی همین مرورگر ذخیره می‌شه — نه توی کد صفحه، و جایی هم فرستاده نمی‌شه مگر وقتی پارتنرت دکمه‌ی ارسال رو بزنه.

از یک توکن محدود (fine-grained) استفاده کن که فقط به همین ریپو و فقط با دسترسی «Contents: Read and write» محدود باشه. هرکسی که به local storage یا Dev Tools این مرورگر دسترسی داشته باشه می‌تونه این رو ببینه — این تنظیمات رو روی سیستم مشترک یا عمومی باز نکن.
